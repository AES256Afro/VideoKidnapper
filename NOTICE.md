## FFmpeg build notes

The **⚙ Setup** dialog's auto-install pulls the `essentials` build from [gyan.dev](https://www.gyan.dev/ffmpeg/builds/), which is distributed under **GPL v3** (linked with GPL components such as x264, libx265, libvpx, etc.).

If you redistribute VideoKidnapper bundled with that FFmpeg build, your distribution must also comply with GPL — notably, offering source for the GPL components on request. For an LGPL-only redistribution, use an LGPL FFmpeg build and adjust the Setup dialog's download URL accordingly.

VideoKidnapper does not redistribute FFmpeg in its source tree or Git history; users install it locally (bundled portable, system package manager, or manually).
